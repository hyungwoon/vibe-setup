# 🚀 바이브 코딩 환경 설치

터미널에 아래 명령어 하나만 입력하면 끝.

```bash
curl -fsSL https://raw.githubusercontent.com/hyungwoon/vibe-setup/main/install.sh | bash
```

## 설치 목록

| 도구 | 용도 |
|------|------|
| **Warp** | AI 터미널 |
| **OpenCode** | 터미널 AI 코딩 에이전트 |
| **Claude Code** | Anthropic AI 코딩 CLI |
| **Git** | 버전 관리 |
| **GitHub CLI** | PR/이슈 관리 |
| **fnm + Node.js LTS** | JS 런타임 |
| **Bun** | 빠른 JS 런타임 & 패키지 매니저 |
| **pnpm** | 빠른 패키지 매니저 |
| **Vercel CLI** | 배포 |
| **Supabase CLI** | 백엔드/DB |
| **Docker** | 컨테이너 |
| **uv** | Python 패키지 매니저 |

## 설치 후

```bash
gh auth login       # GitHub 로그인
vercel login        # Vercel 로그인
supabase login      # Supabase 로그인
claude              # Claude Code 실행
opencode            # OpenCode 실행
```
